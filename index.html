Como Jogar

  1. Toque em uma célula para selecioná-la
  2. Leia a dica que aparece na tela
  3. Use o teclado para inserir letras
  4. Cada letra correta vale 10 pontos
  5. Cada palavra completa vale 50 pontos bônus
  6. Toque em "Mostrar Letra" se precisar de ajuda (custa 30 segundos)
  7. Toque em "Verificar" para conferir suas respostas
  8. Use "Próxima" para avançar para a próxima célula
